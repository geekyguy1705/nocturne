# Personal Website — Nocturne

My personal portfolio site built with **Nocturne** — a dark-first static Astro template. It showcases my blogs, projects, and profile.

## Tech Stack

| Layer | Technology |
|---|---|
| **Framework** | Astro 6 (SSG) + React 19 islands |
| **Styling** | Tailwind CSS v4 — OKLCH custom properties, 15 themes |
| **Animations** | Anime.js v4 — SVG drawables, spring physics |
| **Content** | Markdown content collections, Zod schemas |
| **Search** | Pagefind — post-compile full-text index |
| **Markdown** | KaTeX math, Mermaid diagrams, Shiki syntax highlighting, GFM |
| **UI primitives** | Radix UI, shadcn/ui, cmdk command palette, Lucide icons |
| **Language** | TypeScript 6 |
| **Monorepo** | pnpm workspaces + Turborepo |
| **Fonts** | Inter variable (self-hosted via Fontsource) |
