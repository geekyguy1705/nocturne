# Welcome to Nocturne

This is a demo article running on the **Nocturne 2.0** package.

## Features

- Animated SVG headings
- 15 color themes
- Content collections
- Full-text search via Pagefind
- Mermaid diagram support
- KaTeX math rendering

```mermaid
graph LR
    A[Consumer App] --> B[Nocturne Package]
    B --> C[UI Components]
    B --> D[Layouts]
    B --> E[Routes]
    B --> F[Styles]
```
