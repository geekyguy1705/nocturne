A real-time Kanban board for engineering teams. The app supports drag-and-drop task management, in-card comments, file attachments, and per-project notifications.

## Highlights

- Real-time updates via WebSockets
- Role-based access control
- Optimistic UI for instant feedback
- Automated weekly digest emails

## Outcome

Reduced the average time to close a task by 30% across the client's three engineering squads.
